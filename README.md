Smart Classroom Dashboard

Overview

Smart Classroom Dashboard is an AI-powered classroom monitoring system that automatically tracks teacher attendance 

If an assigned teacher is absent, the system automatically identifies and allocates a suitable substitute teacher based on subject expertise.

The dashboard provides real-time classroom monitoring and alert generation to ensure uninterrupted learning.


Problem Statement

Educational institutions often struggle with:

- Teacher absenteeism
- Delayed substitute allocation
- Manual classroom monitoring
- Communication delays

This system automates these processes using computer vision and intelligent scheduling.

Teacher Presence Detection

Automatic Substitute Assignment
- Subject matching
- Availability checking
- Instant replacement allocation

Classroom Dashboard
